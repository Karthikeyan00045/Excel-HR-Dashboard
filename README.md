# HR Dashboard (Excel)

## 📊 Project Overview
This project analyzes workforce data using Excel pivot tables, slicers, and interactive charts.  
It provides insights into employee demographics, salaries, performance ratings, and HR metrics.

## 📂 Files
- `data table.xlsx` → Raw dataset of 50 employees with demographics, salaries, and performance ratings.
- `HR_Dashboard.xlsx` → Interactive Excel dashboard with slicers and charts.
- `dashboard_screenshot.png` → Preview of the dashboard.

## 🔑 Key Insights
- Gender distribution: 52% Male, 48% Female
- Average performance rating: 2.6
- Highest salary role: HR Specialist
- Remote work: 38% of employees
- Employment status: 40% Part‑Time, 34% Full‑Time, 26% Contract

## 🚀 Features
- Pivot tables for flexible analysis
- Slicers for interactive filtering
- Charts for salary, leave, age, and region breakdowns
- Custom slicer styles for professional polish
